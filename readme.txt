Value Analysis - Value Engineering - Nutzwertanalyse

== Description ==
Value analysis (VA) is a systematic method to improve the "value" of opportunities (goods, products and services) by using requirements. Value analysis is used for the development and improvement of products, technical processes and other operations in all areas of business, science and administration. The application of the value analysis system usually results in a significant improvement and increase in the value of the processed objects, which at the same time is associated with a reduction in effort and costs compared to the original situation. Value analysis is basically carried out as a project and in small interdisciplinary teams.

== Installation ==
1. Deactivate plugin if you have the previous version installed.
2. Extract "value-analysis-nutzwertanalyse.zip" archive content to the "/wp-content/plugins/value-analysis-nutzwertanalyse" directory.
3. Activate "Value Analysis - Nutzwertanalyse" plugin via 'Plugins' menu in WordPress admin menu.
4. create a new value analysis (backend => add => Value Analysis)
5. give the value analysis a name and a small description
6. publish the value analysis
7. go to the frontend page of the article
8. create more users that should participate in the utility analysis in the tab overview
9. create the requirements in the tab utility analysis in any hierarchical form. The requirements can be moved by drag & drop. Save the requirements.
10. create a few options
11. lock the editing of the requirements and the possibilities
Send the Access Token from the Overview tab to the other participants of the utility analysis.
13. weights the requirements in the register utility analysis with the pair comparison method
14. judge how well the possibilities fulfill the requirement
15. interpret the results in the Evaluation tab. Download the results to an Excel list.

5. Gib der Nutzwertanalyse eine treffenedn Namen und eine Kleine Beschreibung

7. Gehe auf die Frontend Seite des Beitrages
8. Erstelle weitere User die an der Nutzwertanalyse teilnehmen sollen im Register Übersicht
9. Erstelle in dem Register Nutzwertanalyse die Anforderungen in beliebig hierarchscher Form. Die Anforderungen können per Drag & Drop verschoben werden. Speichere die Anforderungen.
10. Erstelle ein paar Möglichkeiten
11. Sperre das Editieren der Anforderungen und der Möglichkeiten
12. Versende den weiteren Teilnehmern der Nutzwertanalyse den Access Token aus dem Register Übersicht
13. Gewichte die Anforderungen im Register Nutzwertanalyse mit der Paarvergleichsmethode
14. Beurteile wir gut die Möglichkeiten die Anforderung erfüllt
15. Interpretiere die Resultate im Register Auswertung. Downloade die Resultate in eine Excel Liste.
